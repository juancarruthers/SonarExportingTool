Analyze Projects

To manually analyze projects without any envioronment setted:

  1. Download SonarScanner from SonarCloud website (choose according to your OS).
  2. Download the zip file with the script to automatize the process of uploading multiple projects at the same time.
  3. Decompress the two files downloaded.
  4. Copy all the folders inside the decompressed folder of the SonarScanner (bin, conf, jre and lib) and paste them in the /sonar-scanner subfolder from the Analyze-Projects folder also decompressed in the previous step.
  5. Move the project to be analyzed into the Analyze-Projects/projects subfolder.
  6. Use the sonar-project.properties file inside Analyze-Projects as a template and update the values according to the project to be analyzed, then put it in the project's root directory (the one moved in the previous step). You can find all the parameters of the analysis here (depends on the main language of the project).
  7. Return to the 6th step for each project to be analyzed.
  8. Run the "Analyze-Projects.sh" script. It is a Bash script so you must have Bash installed in order to execute it. If using Windows you can download Git for Windows.

* The errors are logged in "results.txt" file.
