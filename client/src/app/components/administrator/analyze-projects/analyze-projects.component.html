Analyze Projects

To manually analyze projects without any envioronment setted:

  1. Download SonarScanner from SonarCloud website.
  2. Download the zip file with the script to automatize the process to upload many projects.
  3. Decompress the two files downloaded.
  4. Copy all the folders inside the decompressed folder of the sonar-scanner (bin, conf, jre and lib) and paste them in the sonar-scanner subfolder from the analyzeProjects folder.
  5. Move the projects that are going to be analyzed in the /projects subfolder inside the analyzeProjects one.
  6. Use the sonar-project.properties file as a template and update the values according to the project and then put it in the project's root directory. You can search for all the analysis parameters (depends in the language) here.
  7. Repeat the 6th step for every project that is going to be analyzed.
  8. Run the "anlyzeProjects.sh" script.

* If an error appears with a project you can check it in "results.txt" file.
